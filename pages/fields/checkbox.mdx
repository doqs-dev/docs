# Checkbox field

Add a checkbox field and control on which condition it should be checked. The checkbox style is currently limited to a cross.

### Attributes
<Grid>
    <div>
        <Callout>
            The field is rendered as a square. The bounding box height is determining the size of the square.
            The square is centered horizontally in the bounding box.
        </Callout>

        <CommonFields/>

        <Attribute name="type" required="1">Identifier of the type of the field. Always set to `check`.</Attribute>

        <Attribute name={"eval"} required="1">
            String or boolean value to evaluate against if the checkbox is checked. E.g. the check box is checked if the
            equal boolean value is provided or if the string is matching with te value of the field.
            A string must be longer than 1 character and the evaluation is case sensitive.
        </Attribute>
    </div>
    <div className="relative">
        <div className="sm:sticky top-0">
            <SimpleCodeHighlighter language="json" code={`{
    "bbox": {
        "x": 72.15405575993107,
        "y": 556.1774688664345,
        "width": 201.3371400656083,
        "height": 11.637984974890685
    },
    "name": "key",
    "page": 0,
    "required": true,
    "type": "check",
    "eval": true
}`}/>
        </div>
    </div>
</Grid>
