# Date field

Add a date / time or datetime field to your template and control the output format with the `format` and `locale` attributes.

### Attributes
<Grid>
    <div>
        <Attribute name="bbox.x" required="1">
            X coordinate of the lower left corner of the bounding box of the field
        </Attribute>
        <Attribute name="bbox.y" required="1">
            Y coordinate of the lower left corner of the bounding box of the field
        </Attribute>
        <Attribute name="bbox.width" required="1">
            Width of the bounding box of the field
        </Attribute>
        <Attribute name="bbox.height" required="1">
            Height of the bounding box of the field
        </Attribute>

        <Attribute name="name" required="1">
            Key name of the field. This is going to be the key for populating the value when
            [filling](/api-reference/fill-template) the template. This key name can be used multiple times, every
            occurrence of the key will be filled with the same value. Caution: do not mix field types.
        </Attribute>

        <Attribute name="page" required="1">The index of the page the field is residing on, starting with 0. Upper limit
            is the number of pages in the template</Attribute>
        <Attribute name="type" required="1">Identifier of the type of the field. Always set to `date`.</Attribute>

        <Attribute name="font_size" required={false}>Specify the font size to be filled with. Defaults to 9</Attribute>
        <Attribute name="color" required={false}>Specify the font color to be used. Format is either common color cod
            names (e.g. black, red, etc.) or hex color codes. Defaults to `black`.</Attribute>

        <Attribute name="h_align" required={false}>Horizontal alignment of the to be filled text.
            Possible values are `L` for left aligned, `C` for centering and `R` for right aligned text. Defaults to
            `L`.</Attribute>
        <Attribute name="v_align" required={false}>Vertical alignment of the to be filled text.
            Possible values are `T` for top aligned, `C` for centering and `B` for bottom aligned text. Defaults to
            `T`.</Attribute>

        <Attribute name="locale" required={true}>The locale which should be used for formatting the
            datetime.</Attribute>
        <Attribute name="format" required={true}>
            Format for the resulting datetime. Examples for date '2022-05-21 21:16:00+02:00':
            * `%Y-%m-%d` will output `2022-05-21`
            * `%d-%m-%Y %H:%M:%S` will output `21-05-2022 21:16:00`
            * `%A` outputs the full week day in the specified locale, e.g with locale `en_US`: `Saturday`
            * `%x` outputs locale’s appropriate date representation, e.g. with locale `en_US`: `05/21/2022`
            * `%X` outputs locale’s appropriate time representation, e.g. with locale `en_US`: `21:16:00`
            * `%B` outputs locale’s appropriate month name, e.g. with locale `en_US`: `May`
        </Attribute>
