# Create a template

<Endpoint method="POST" path="/template"/>
<br/>

For [authentication](/api-reference/authentication)  provide `x-api-key` header.

<br/>
## Request
<Grid>
    <div>
        ### Header parameters
        <Attribute name="Content-Type" required={true}>The content type header needs to be set to `multipart/form-data`</Attribute>
        <br/>

        ### Body
        * `file`: [required] multipart binary pdf file
        <br/>
    </div>
    <SimpleCodeHighlighter language="python"
                           code={`import requests
with open('my-template.pdf, 'rb) as pdf:
    requests.post("https://api.doqs.dev/v1/templates/",
        headers={
            "x-api-key": "your-api-key",
            "content-type": "multipart/form-data"
        },
        files={"file": ("my-template.pdf", pdf)}
    )
`}
    />
</Grid>


<br/>

## Response

<Grid>
    <div>
        Body attributes are nested inside the `results` to align with the other endpoints.
        ## Body attributes

        * `id`: Unique identifier for the template
        * `name`: Name of the template, this is prefilled from the filename of the uploaded file.
        * `fields`: List of fields in the template. Always empty for now (Planned feature: automatically extracting and adding fields from the template)

    </div>

    <SimpleCodeHighlighter language="json"
                           code={`{
    "results: {
        "id": "tpl_d7d84644c9374cdf83e2c8463279e23c",
        "name": "my-template",
        "fields": []
    }
}
`}/>
</Grid>
