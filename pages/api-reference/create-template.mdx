# Create a template

<Endpoint method="POST" path="/template"/>
<br/>

For [authentication](/api-reference/authentication)  provide `x-api-key` header.


## Request
<Grid>
    <div>
        ### Header parameters
        * `x-api-key`: [required] API key
        * `Content-Type`: [required] multipart/form-data
        <br/>

        ### Body
        * `file`: [required] multipart binary pdf file
        <br/>
    </div>
    <SimpleCodeHighlighter language="python"
                           code={`import requests
with open('my-template.pdf, 'rb) as pdf:
    requests.post("https://api.doqs.dev/v1/templates/",
        headers={
            "x-api-key": "your-api-key",
            "content-type": "multipart/form-data"
        },
        files={"file": ("my-template.pdf", pdf)}
    )
`}
    />
</Grid>


<br/>
