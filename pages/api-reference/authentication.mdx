# Authentication

Authenticate your api calls by providing your api key in the header. You can retrieve and manage your api keys in the [dashboard](https://app.doqs.dev/).

We are offering two API keys: one development and a live one. The live api key is only available after you subscribed.

## Headers

| Header    | Value                           |
|:----------|:--------------------------------|
| x-api-key | your api key from the dashboard |


## Examples
This is an example of how to authenticate your api calls in python with the requests library.
<SimpleCodeHighlighter language="python" code={`import requests
requests.post("${baseUrl}/templates", headers={"x-api-key": "your api key"})
`}
