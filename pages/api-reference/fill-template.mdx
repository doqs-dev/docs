# Fill template

## Request
<Grid>
    <div>
        ### Header attributes
        <Attribute name={"x-test"} required={false}>
            Control if this request should be counted as a sandbox request or not. If your organization is not
            subscribed yet this flag is set to `true` by default.
        </Attribute>
        <Attribute name={"content-type"} required={true}>
            Only acceptable content type is `application/json`.
        </Attribute>

        ### Body attributes
    </div>
    <SimpleCodeHighlighter language="python" code={`import requests
