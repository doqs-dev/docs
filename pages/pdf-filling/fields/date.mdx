# Date field

Add a date / time or datetime field to your template and control the output format with the `format` and `locale` attributes.

### Attributes
<Grid>
    <div>
        <CommonFields/>

        <Attribute name="type" required="1">Identifier of the type of the field. Always set to `date`.</Attribute>

        <TextAttributes />

        <Attribute name="locale" required={true}>The locale which should be used for formatting the
            datetime.</Attribute>
        <Attribute name="format" required={true}>
            Format for the resulting datetime. Examples for date '2022-05-21 21:16:00+02:00':
            * `%Y-%m-%d` will output `2022-05-21`
            * `%d-%m-%Y %H:%M:%S` will output `21-05-2022 21:16:00`
            * `%A` outputs the full week day in the specified locale, e.g with locale `en_US`: `Saturday`
            * `%x` outputs locale’s appropriate date representation, e.g. with locale `en_US`: `05/21/2022`
            * `%X` outputs locale’s appropriate time representation, e.g. with locale `en_US`: `21:16:00`
            * `%B` outputs locale’s appropriate month name, e.g. with locale `en_US`: `May`
        </Attribute>
